Praktek Pemrograman Web 1

Sesi 1

praktek pemrograman Web 1 akan membahas tentang cara menampilkna halaman Web dengan menggunakan HTML, CSS, dan Javascript. Mahasiswa diharapkan mampu memahami sintaks dasaar dalam menampilkan halaman web, dan dengan pemahaman tersebut menuliskan hasil kreasinya sendiri.

Pada sesi 1 mahasiswa mengenali dulu truktur dasar HTML beserta sintaks dasarnya.

  • HTML
  • CSS
  • Javascript

Pada sesi 1 mahasiswa mengenali data struktur dasar HTML beserta sintaks dasarnya.

  1. Pengenalan HTML
  2. Tabel
  3. Form
